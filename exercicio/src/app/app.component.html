Exercício 001 - em Angular

O objetivo deste exercício é familiarizar-se com a manipulação de dados no template de um componente Angular. Será necessário criar um componente e realizar operações básicas, como exibir dados, interagir com eventos e atualizar o conteúdo dinamicamente.

Instruções:

  1. Crie um novo componente chamado DataManipulationComponent.
  2. Defina um array de objetos que representem dados fictícios. Cada objeto deve ter pelo menos três propriedades.
  3. No template do componente, utilize estruturas de repetição para exibir os dados em uma lista.
  4. Adicione um botão ou elemento interativo para cada item na lista. Ao clicar no botão, exiba no console uma mensagem ou informações específicas sobre o item relacionado.
  5. Adicione um botao para excluir o item.
