Ex 002 - Compatilhamento de dados entre componente pai para o filho

Você foi designado para criar um aplicativo Angular que demonstre a comunicação de dados entre um componente pai e um componente filho. O objetivo é passar informações do componente pai para o componente filho e exibi-las na interface do usuário.

Tarefas:

  • Crie um componente chamado "PaiComponent" que contenha um campo de entrada (input) e um botão.
  • Crie um segundo componente chamado FilhoComponent que exibirá os dados recebidos do componente pai.
  • Configure um serviço para gerenciar a comunicação entre os componentes. Este serviço será responsável por transmitir os dados do "PaiComponent" para o "FilhoComponent".
  • Ao pressionar o botão no PaiComponent, os dados inseridos no campo de entrada devem ser transmitidos ao "FilhoComponent"
  • Exiba os dados recebidos no FilhoComponent em uma área específica da interface

Requisitos adicionais:

  • Utilize as práticas recomendadas do Angular para a comunicação entre componentes.
  • Certifique-se de que os dados são atualizados dinamicamente no FilhoComponent quando o conteúdo do PaiComponent é alterado.
  • Adicione uma funcionalidade de reset no FilhoComponent para limpar os dados exibidos.
